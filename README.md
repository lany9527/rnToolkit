# rnToolkit

![GitHub](https://img.shields.io/github/license/mashape/apistatus.svg)

rnToolkit 是移动端 React Native 实现，提供了基础的 UI 组件以及一些常用的业务组件。

我们的目标是让 React Native 项目开发更快、更简单。
